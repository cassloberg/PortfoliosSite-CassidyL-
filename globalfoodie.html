Global Foodie

  • Duration: 2 months
  • Role: UI Designer
  • Objective: Design a responsive e-commerce application
  • Tools: Adobe XD, Adobe Illustrator, Adobe Photoshop

Project Background

I created this project as part of the CareerFoundry UI Design course

App requirements
  • customers should be able to save items to a wishlist
  • customers should be able to filter products
  • customers should see recommended products
  • customers should be able to create an account
  • customers should be able to shop with or without an account

I decided to create Global Foodie's app concept because my significant other always talks about snacks from his home country, and I thought "how great would it be if we could just order them!?"

Design Process

I began my design on paper, mapping out a general idea of how I wanted the design to look and navigate.

As I transitioned to digital wireframes,I kept things simple, focusing on the user flow and experience.

When creating my final design, I added vibrant colors and imagery. I attempted to integate the mascot into more screens, as users seemed to have taken a liking to him.
